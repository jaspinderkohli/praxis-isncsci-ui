# Praxis ISNCSCI classification

## Design Problem

We need a section to be use as a dialog or docked at the bottom of the screen to capture the ISNCSCI classification.
Contains the values used to determine the level and severity of the injury and to monitor neurological recovery.

## Approach

The section is composed of a header and a body. The header contains a title and a close button.
The body contains the different sections for the classification values.
We give it an acrylic material look to make it look like it's floating on top of the content.
We use composition and provide slots for the dirrerent sub-sections.

### Slots

- `header` - Contains the value to be displayed in the dialog's header
- `neurological-levels` - Section for the neurological levels
- `nli` - Section for the Neurological Level of Injury
- `ais` - Section for the Asia Impairment Scale and injury completeness
- `zpp` - Section for the Zone of Partial Preservation
- `sub-scores` - Section for the sub-scores
