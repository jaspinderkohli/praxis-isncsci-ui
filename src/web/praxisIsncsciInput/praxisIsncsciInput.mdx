# Praxis ISNCSCI Input

## Design Problem

An imput component that docks at the top of the input layout on large screens and is fixed to the bottom of the input layout on small screens.

It is made of two sections:

- A grid of buttons to input sensory and motor values.
- Input controls to capture extra information when using the _star (\*)_ flag. This section should be hidden when the _star (\*)_ flag is not selected.

## Approach

We use a flex layout to easily switch from row, when on large screens, to column, when on small screens.
For the button section we use a grid layout to easily align the buttons.
