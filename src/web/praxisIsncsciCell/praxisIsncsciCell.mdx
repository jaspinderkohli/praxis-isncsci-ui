# Praxis ISNCSCI Cell

## Design Problem

We need a cell component to compose our ISNCSCI table.

## Approach

We are favoring composition so we add slots for the optional subcomponents.
This will make it easy for the component or controller adding them to control their appearance and subscribe to their events.

### Slots

- There is a deffault slot for the cell content.

### Attributes

- `error` - boolean - if true, the error style is applied to the cell
- `highlighted` - boolean - if true, the cell will be highlighted
